# Модель предсказания финансирования НКО

## Описание проекта
Данное приложение предсказывает вероятность получения 
финансирования НКО на основе характеристик организации.

## Технологии
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## Что делает приложение
- Принимает данные об НКО
- Анализирует характеристики организации
- Предсказывает вероятность получения финансирования
- Показывает важность признаков
